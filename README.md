# GPS・写真・Excel 照合ツール

放射線移動モニタリング後の照合作業（目視・手作業で約1時間）を効率化するPC用Webアプリです。

## 起動方法

```bash
npm install
npm run dev
# → http://localhost:5173 をブラウザで開く
```

Windows PC の Chrome / Edge で動作確認しています（画面幅 1200px 以上推奨）。

---

## 画面構成

### 画面1 — ファイル読み込み

- 今回・昨年の写真（ZIP またはフォルダ）と Excel ファイル 4 点をドラッグ＆ドロップで読み込み
- 写真ファイル名の命名規則: `{地点番号}a.jpg` / `{地点番号}b.jpg`（例: `001a.jpg`, `001b.jpg`）
- 読み込み後に「a写真不足・昨年写真なし・Excel不一致の地点」を即座に表示
- Excel の列名を自動検出。検出失敗時は手動マッピング UI を表示
- HEIC ファイルは未対応（jpg/jpeg への変換を案内するメッセージを表示）

### 画面2 — チェック結果一覧

横スクロール対応の表形式で以下を表示します。

| 列 | 内容 |
|---|---|
| 地点番号 | 固定列 |
| 今回a / b 写真の有無 | ✓ / ✗ |
| 昨年a / b 写真の有無 | ✓ / ✗ |
| 写真ファイルチェック | OK / NG / 要確認 |
| 写真GPS 北緯・東経 | EXIF から取得した度分秒 |
| 今回Excel 北緯・東経 | Excel の GPS 値 |
| 昨年Excel 北緯・東経 | 昨年 Excel の GPS 値 |
| 写真↔ExcelGPS照合 | OK / NG / 要確認 |
| 北緯・東経 差分(秒) | 今回と昨年の差（総秒数で計算） |
| GPS 判定 | OK / 要確認 / NG |
| 測定値 | Excel から表示（自動判定なし） |
| 写真確認結果 | インラインセレクタ（人間が入力） |
| メモ | 自由記述 |
| 総合判定 | OK / NG / 要確認 |

**操作機能**
- OK / NG / 要確認 でフィルター
- 地点番号で検索
- NG 行: 赤背景、要確認行: 黄色背景
- Excel 出力（色付きセル）・テキストメモ出力

### 画面3 — 写真横並び確認

PC の横長画面を活かした 2×2 グリッドで表示します。

```
[今回 001a（看板）]   [昨年 001a（看板）]
[今回 001b（全景）]   [昨年 001b（全景）]
```

**操作機能**
- ← → キーで地点移動
- 数字キー `1`: OK、`2`: NG、`3`: 要確認
- 写真クリックで全画面拡大（Esc で閉じる）
- メモ入力欄

## GPS 照合ロジック

### 写真 GPS と今回 Excel GPS の照合

1. 写真 EXIF から GPS を取得（小数度）
2. 度分秒に変換し、秒を小数第2位に四捨五入
3. Excel の値と比較（許容差: 0.05 秒 = 丸め誤差のみ）

### 今回 Excel と昨年 Excel の GPS 差分

```
総秒数 = 度 × 3600 + 分 × 60 + 秒
```

60 進法の桁をまたぐ誤判定を防ぐため、必ず総秒数に変換してから比較します。

| 差分 | 判定 |
|---|---|
| ±5 秒以内 | OK |
| ±10 秒以内 | 要確認 |
| ±10 秒超 | NG |

**例**: 昨年 37°45'59" → 今回 37°46'01" = 差分 **2 秒**（59→01 の誤判定なし）

## 出力ファイル

### チェック結果付き Excel

元の今回測定記録 Excel をベースに、以下の列を追加して出力します。

- 写真ファイルチェック
- 写真 GPS 照合結果
- 昨年 GPS 比較結果
- 北緯・東経 差分秒
- 写真確認結果
- メモ
- 総合判定

NG セル: 赤背景、要確認セル: 黄色背景（xlsx-js-style による色付け）

### 上司共有用テキストメモ

NG・要確認の地点だけを抽出してテキストファイルで出力します。

## 技術スタック

| ライブラリ | 用途 |
|---|---|
| Vite + React 18 | フレームワーク |
| Tailwind CSS 3 | スタイリング |
| xlsx-js-style | Excel 読み書き + 色付け |
| exifr | 写真 EXIF GPS 取得 |
| jszip | ZIP ファイル展開 |

## 対応環境

- Windows 10/11 の Chrome / Edge（最新版）
- 画面幅 1200px 以上
- ブラウザ内で完結（サーバー不要・外部送信なし）
